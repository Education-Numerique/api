Support¶

If you have a questions or issues about Lxxl, there are several options:

Send a Tweet¶

If your question is less than 140 characters, feel free to send a tweet to @lxxl.

File an Issue¶

If you notice some unexpected behavior in Lxxl, or want to see support for a new feature, file an issue on GitHub.

E-mail¶

I’m more than happy to answer any personal or in-depth questions about Lxxl. Feel free to email hello@education-numerique.com.
